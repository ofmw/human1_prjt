인터셉터(interceptor)
- Spring에서는 HandlerInterceptor인터페이스를 구현한 클래스로 사용자의 요청을 컨트롤러가 실행하기 전
  컨트롤러가 실행한 후, 뷰를 실행한 후에 실행의 흐름을 가로채서 공통기능을 적용시킬 수 있도록 지원함
- HandlerInterceptor인터페이스에서는 JDK8 이후부터 3개의 메소드를 디폴트메소드로 정의해두고 지원함
  : preHandle(), postHandle(), afterCompletion() 이들 가운데 필요한 메소드를 Override해서 사용함

- 로그인 인터셉터 구현 및 적용
  : 회원만 보낼 수 있는 요청URL이 주소표시줄을 통해서 입력될때 컨트롤러가 요청을 실행하기 전에 가로채어
    로그인 화면이 출력되도록 함
- 사용자의 정상적인 요청URL에 대해서 가로챔
